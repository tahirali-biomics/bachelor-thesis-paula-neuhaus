# Bachelor Thesis Template

This repository provides a structured template for bachelor thesis projects.

## Structure

- `thesis/` – Quarto thesis files, references, figures and tables
- `scripts/` – analysis scripts in R, Python and Shell
- `data/` – raw, processed and metadata files
- `results/` – figures, tables and supplementary outputs
- `docs/` – additional documentation
- `environment/` – reproducibility files
